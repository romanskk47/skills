# Claude Code Skills

A collection of skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) - Anthropic's official CLI for Claude.

## Available Skills

| Skill | Description |
|-------|-------------|
| [x-engagement-guide](./x-engagement-guide) | Maximize X (Twitter) engagement based on reverse-engineering the open-sourced algorithm |

## Installation

Clone and copy to your Claude Code skills directory:

```bash
git clone https://github.com/romanskk47/skills.git
cp -r skills/x-engagement-guide ~/.claude/skills/
```

## What are Skills?

Skills are modular packages that extend Claude Code's capabilities with specialized knowledge and workflows. They activate automatically based on context - just ask Claude about the topic and the skill kicks in.
